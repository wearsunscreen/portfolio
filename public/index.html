My current project is Viple.

Learn vi by playing classic games.

Play Here!

Motivation: While some devs choose to learn vi (I'm not judging) I recently discovered that some students are forced to learn vi. There are a few games available to help you learn some features of vi, there's a need to teach more.

Technology: I wanted to learn Go and I was not disappointed. Go is designed with the developer experience in mind. It's simple and powerful. The ebitengine is a solid 2D game dev framework.

Checkout the code at https://gitlab.com/johncrane/viple

Gopher contributions welcome!

    Levels so far:
  • Flappy bird - J, K - move up and down
  • Brickout - H, L - move left and right
  • Four edge Brickout - H, J, K, L
  • Connect 3 - Delete lines
  • Connect 3 - Visual Mode
  • Connect 3 - Challenge level, use all skills
  • Connect 3 J, K - move left and right - Brickout
In progress: Snake - HJKL (replace Four Edge Brickout)
